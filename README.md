<h2> ✓ 개요</h2>

1. 프로젝트 소개<br>
   ◦ Java 언어로 게임을 만들고 서버와 클리어은트를 나눠 방명록, <br>
   친구 수집을 진행하며 swing을 사용하여 gui를 구현하고, tcp 통신으로 채팅하는 동물의 숲 모방 게임

   ◦ 5인
5. 담당역할<br>
   ◦ 친구 추가, 게임(화석발굴,너굴맨잡기,컴퓨터를 이겨라) 개발 담당
